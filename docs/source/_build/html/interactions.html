Slash-command constructor¶

This tool allows to build slash-commands for further registration. Registering a new command is required due to the way Discord parses slash-commands.

1. User inputs data
2. Discord converts this data into command args
3. Discord API sends converted data to your app

The second step is never completed if your command isn’t registered. There’re 2 types of slash-commands: global and local (per guild).

Note

Global command registration takes more than 1 hour.
Guild command registration is instant.
